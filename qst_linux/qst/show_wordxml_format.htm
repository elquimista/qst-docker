Only multiple choice - single answer questions can be imported. (15 answers max.)

Open your word .doc and make sure you put QST as the first word at the beginning of each question.

Each answer should have )@ at the begining of the line with each answer.

Each correct answer should have ** as the first letters of the answer.

The characters )@ and ** and QST should not be used in any questions or answers as they are special characters in the import format.

Once you have done the above, save your word doc as xml and import it.

An example:

QST What is the bird below?

   

)@ duck
)@ owl
)@
)@ ** pheasant
